# column-addition-cpp
簡単な+-筆算シュミレーター

## Environment
- apple m2
- macOS Sonoma 14.0
- cmake min:3.22.1 (current:3.27.7)
- c++20~
- g++13
  
## How to run
 `cmake` or `docker` or `Visual Studio 2022`

### in Visual Studio 2022 (recommended)
- open Visual Studio 2022
- run on `Debug` mode

### cmake
```bash
make run
```



### docker (not recommended...)
恐らくセットアップを書き直さないと標準入力使用不可
```
make docker
```

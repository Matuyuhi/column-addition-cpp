# column-addition-cpp

### Description

簡単な筆算シュミレーター

### Environment
- apple m2
- macOS Sonoma 14.0
- cmake 3.22.1
  
### How to run
 `cmake` or `docker` or `Visual Studio 2022`

 - cmake
```bash
make run
```
- docker
```
make docker
```

- in Visual Studio 2022
  - open Visual Studio 2022
  - run on `Debug` mode
